Loading and preprocessing the data

First unzip the “activity.zip” file.

unzip('activity.zip')

Next read the “activity.csv” file into a data.table using the fread call. This data.table will serve as the base for the remainder of the study.

library(data.table)
# Read the data into a data.table
activity <- fread( 'activity.csv',
        header=TRUE, sep=",", na.strings = "NA",
        colClasses=c('numeric', 'Date', 'numeric')
    )
# Define the indexing key on the data.table
setkey(activity, date)

  • What is the impact of inputting missing data on the estimates of the total daily steps? First and most obviously any days with NA present in their record has an automatic increase to their values. This is most visual in the first few days where they're largely NA data.
